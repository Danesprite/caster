Caster

Dragonfly-Based Voice Programming Toolkit

View on GitHub Download .zip Download .tar.gz

Caster is a collection of tools aimed at enabling programming entirely by voice. It runs on top of Dragonfly, which runs on either Dragon NaturallySpeaking or Windows Speech Recognition.
